Время работы ноутбука ~ 34 минуты.

Решение идейно:
Возьмём базовое решение с подсчитанными в нём параметрами (параметр для первого товара был подсчитан мной более точно, а для для всех - не хватило времени), вторым решением будет прогноз линейной регрессии с L1-регуляризатором (Lasso) по признакам: значения ряда на некотором промежутке в прошлом, день недели (как 7 бинарных признаков (надо было шесть, но уже всё равно)). 

Напишем простенький валидатор, считающий SMAPE, при прогнозах, когда от имеющихся 304 значений для каждого товара берём сколько-то первых (< 275) и пытаемся делать прогноз на месяц (30) вперёд, после чего считаем SMAPE на прогнозе и данных, которые у нас есть.

По такому качеству на валидации дополнительно определим, какой промежуток в прошлом задаёт лучший набор признаков (типа прошлые две недели или позапрошлая и позапозапрошлая недели) для каждого товара при прогнозировании с помощью Lasso.

После этого, по качеству на валидации, для каждого товара определим, чем лучше его прогнозировать: c помощью SARIMAX с параметрами, которые фиксированы и выбраны заранее или c помощью Lasso.

Спрогнозируем каждый товар выбранной моделью.
